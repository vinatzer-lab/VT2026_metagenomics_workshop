# VT2026_metagenomics_workshop
VT 2026 workshop for metagenomics showcasing the use of GenomeRxiv and VT-PLANS. Repository contains all input files used in the demonstrations.
***Note: Please come to the workshop with a bacterial genome of interest to analyze (NCBI accession or FastA file)*** 

## Workshop Outline
### Morning: GenomeRxiv 
- Introduction to GenomeRxiv
- Hands on session of precise pathogen identification and analysis using the [GenomeRxiv web server](http://genomerxiv.org).
    - Demonstration of taxonomic circumscription
- Demonstration of [BrookLIN](https://code.vt.edu/linbaseproject/brooklin) the commandline interface for GenomeRxiv. 

### Afternoon: VT-PLANS
- Introduction to VT-PLANS and sample submission
- Demonstration of pathogen reporting
- Tour of the database/management system
